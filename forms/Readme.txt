⚙️ CONFIGURACIÓN DEL FORMULARIO DE CONTACTO
-------------------------------------------

1. Copia el archivo `.env.example` y renómbralo como `.env`.
2. Abre el nuevo archivo `.env` y reemplaza:
   - TU_API_KEY_AQUI → por tu clave real de SendGrid.
   - EMAIL_FROM → por el correo desde el que se enviarán los mensajes.
   - EMAIL_TO → por el correo donde deseas recibir los mensajes.
3. Asegúrate de tener instaladas las dependencias ejecutando:
   composer install
4. Ejecuta el proyecto desde tu servidor local PHP o desde tu hosting.
